# Controles

## Caméra

Rotation de la caméra : souris

Zoom/Dezoom : molette (distance:vue exterieure / FOV:vue interieure)

Vue FPS/TPS : V

## Avion

### General

Batteries : N/A

Activer / Désactiver l'APU : P

Debloquer la manette des gaz : I

### Axes

**Joystick**

Pitch/Yaw : stick gauche

Throttle/Roll : stick droit

**Trims**

Pitch : S/Z

Yaw : D/Q

Roll : E/A

Throttle : haut/bas
